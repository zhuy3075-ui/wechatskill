# wechat-writer（对外发布版）

面向微信公众号创作的 AI skill，支持写作、风格学习、素材沉淀、质量闸门与发布前合规检查。

## 你能做什么

1. 写五类文章：`干货 / 观点 / 故事 / 清单 / 热点`
2. 学作者风格：输入范文后生成 `styles/[作者名].md`
3. 自动质检：原创度、AI 味、人味三项打分
4. 管理素材库：支持 `M-XXX` 主键和 CSV 导出
5. 输出成品：`md / json / both / wechat`

## 快速开始

直接用自然语言：

- `写一篇观点文章，主题是AI对教育的影响`
- `把这段素材写成故事型公众号文章`
- `先推荐最适合的风格，再开始写`
- `输出公众号可直接粘贴版本`

也可用命令式：

- `/wechat-writer 干货 时间管理`
- `/wechat-writer 观点 AI对教育的影响 --style 理性派`
- `/wechat-writer 输出格式 wechat`

## 关键规则（简版）

1. `--style` 只改表达方式，不改人格与底线。
2. 学习范文必须提供作者名/账号名。
3. 全局可读性规则始终生效；各风格偏好互相独立。
4. 成品默认需过闸门：
   - `originality_score >= 70`
   - `ai_tone_score <= 30`
   - `humanity_score >= 60`
5. 不达标时优先“定向修复”，不是直接整篇重写。

## 输出模式

- `md`：Markdown 发布稿
- `json`：结构化结果
- `both`：`md + json`
- `wechat`：公众号编辑器直贴纯文本

## 分发前提醒

发布前请确认仓库不含以下本地残留：

- `.claude/`
- `output_draft.md`
- 个人化风格档案（按需保留）

## 文档入口

- 用户操作手册：`USER_GUIDE.md`
- 详细规则：`SKILL.md`
- 提示词库：`prompt.md`
- 开发维护版：`README.dev.md`
