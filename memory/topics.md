# 选题库

> 选题库采用 CSV 格式管理，支持按赛道分类保存、去重合并、爆款潜力评分。
> 来源：对标文章提取 / 已完成文章归档 / 用户灵感 / 热点事件。
> 详细提示词见 [prompt.md](../prompt.md) 第 7.7 节。
>
> **数据存储说明**：本文件是选题库的「规范定义 + Markdown 备份」。
> 完整 CSV 数据由 AI 在对话中生成并输出，用户自行保存为 .csv 文件。
> 本文件的「选题库档案」区域保留关键记录的 Markdown 备份，用于 AI 跨会话查阅。

## CSV 格式规范

文件命名：`选题库_[来源或主题]_[YYYY-MM-DD].csv`
编码：UTF-8 BOM（Excel 兼容）

### 列定义

```csv
选题名称,一级赛道,二级赛道,文章类型,爆款潜力(1-10),开幕雷击要素,目标读者,情绪基调,切入角度,来源,来源链接,关键词,状态,添加日期,备注
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 选题名称 | 一句话描述选题方向（不是标题） | 35岁程序员转行做自媒体的真实经历 |
| 一级赛道 | 主分类 | 职场成长 |
| 二级赛道 | 子分类 | 副业赚钱 |
| 文章类型 | 干货/故事/观点/清单/热点，可多选用"+"连接 | 故事+观点 |
| 爆款潜力 | 1-10 分综合评估 | 8 |
| 开幕雷击要素 | 可用的要素 | 金钱+异常+悬念 |
| 目标读者 | 画像描述 | 25-35岁职场人 |
| 情绪基调 | 文章情绪方向 | 励志振奋 |
| 切入角度 | 建议的写作切入点 | 从月薪3万到月入800的落差切入 |
| 来源 | 对标文章/自主生成/用户提供/热点事件 | 对标文章 |
| 来源链接 | URL（如有） | https://... |
| 关键词 | 3-5 个 | 转行,自媒体,程序员,中年危机 |
| 状态 | 待写/已写/已弃/进行中 | 待写 |
| 添加日期 | YYYY-MM-DD | 2025-01-15 |
| 备注 | 效果数据、补充说明等 | 阅读量12000 |
