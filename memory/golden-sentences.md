# 金句库

> 收集每次创作中产生的优质金句，按主题分类，供后续文章复用。

## 使用说明
- 每次创作后，将文章中的精彩金句摘录于此
- 按主题分类，标注出处文章
- 标记使用次数，避免高频重复使用同一金句
- 超过 100 条时，淘汰使用效果不佳的
- 写作前先读 `memory/materials.md` 的「金句观点/类比比喻」分区，从高质量素材（质量分>=4）提炼表达模式
- 金句允许”借鉴表达框架”，但必须重写为原创表述，不得直接照抄素材原句
- 如某条金句受素材启发，可在行尾追加 `#来源素材 M-XXX` 方便回溯

## 金句格式（含风格标签）

每条金句必须包含风格标签，用于按风格检索和学习：

```
- [G-001] 「金句内容」
  出自：《文章标题》(日期)
  风格：犀利 / 温暖 / 幽默 / 哲理 / 朴实 / 激励 / 反讽（可多选，用+连接）
  句式：对比式 / 反转式 / 定义式 / 条件式 / 排比式 / 类比式 / 反问式
  话题：[所属话题]
  已用：N 次
  关联观点：V-XXX（如有对应观点）
```

**金句ID 规则**：`G-XXX`，三位递增编号，删除后不回收。

## 风格标签体系

| 风格标签 | 特征描述 | 典型句式 |
|---------|---------|---------|
| 犀利 | 一针见血、直击本质、不留情面 | “所谓XX，不过是YY的遮羞布” |
| 温暖 | 治愈、共情、柔软有力量 | “没关系，XX本来就是YY的过程” |
| 幽默 | 轻松、自嘲、反差萌 | “XX这件事，说白了就是YY” |
| 哲理 | 深度思考、洞察本质、引人深思 | “XX的反面不是YY，而是ZZ” |
| 朴实 | 大白话、接地气、真实感 | “说到底，XX就是YY” |
| 激励 | 鼓舞、正能量、行动导向 | “与其XX，不如YY” |
| 反讽 | 反话正说、黑色幽默、社会批判 | “XX当然是对的，毕竟YY嘛” |

## 创作时金句策略

> 生成文章时，金句的来源有两种路径，根据场景选择最优方案。

### 路径一：直接复用旧金句

**适用场景**：
- 金句库中有与当前文章话题高度匹配的金句
- 该金句使用次数 < 3 次（避免读者审美疲劳）
- 金句的风格与当前文章风格一致

**复用规则**：
- 优先复用质量高（被保留次数多）且使用次数少的金句
- 可以对旧金句做微调（换个主语、调整语序），但保留核心表达
- 复用后更新使用次数

### 路径二：学习风格生成新金句

**适用场景**：
- 金句库中没有匹配的现成金句
- 当前文章需要特定风格的金句（如犀利风格的文章需要犀利金句）
- 用户指定了风格（如 `--style 犀利`）

**生成流程**：
1. 确定目标风格标签（从文章风格或用户指定中获取）
2. 从金句库中筛选同风格的金句 3-5 条，分析其共同句式特征
3. 从 `materials.md` 第三节和第六节中筛选同风格的外部金句，提炼表达模式
4. 如有对应的 `styles/[风格名].md`，参考其「金句模式」章节
5. 基于以上分析，生成 3 条候选金句
6. 选择最贴合文章语境的 1-2 条使用
7. 使用后的新金句自动入库（标注风格标签和句式）

### 路径选择决策树

```
需要金句
  ├─ 金句库有匹配的？
  │   ├─ 是 → 风格一致？
  │   │       ├─ 是 → 使用次数 < 3？
  │   │       │       ├─ 是 → 路径一：直接复用
  │   │       │       └─ 否 → 路径二：参考该金句风格生成新的
  │   │       └─ 否 → 路径二：按当前文章风格生成
  │   └─ 否 → 路径二：按当前文章风格生成
  └─ 生成后 → 新金句入库（含风格标签）
```

## 自学习规则（来自素材库）
- 触发时机：每次写作前、每次范文学习后、每次用户提供文章后
- 学习对象：`materials.md` 第三节（金句观点）与第六节（类比比喻）
- 学习动作：
  - 提炼高频句式（如”反差句式””递进句式”）
  - 提炼高复用修辞（如类比、对照、反问）
  - 按风格标签分类归纳句式模式
  - 生成 3-5 条新金句候选，再按文意筛选入库
- 质量门槛：仅学习质量分 `>=4` 且来源清晰的素材
- 防污染：若与 `core/personality.md` 当前语气冲突，保留技巧不用原语气
