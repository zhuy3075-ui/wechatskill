## 二、高效写作模式（自动提炼）

> 积累 10+ 篇效果数据后，系统自动分析以下维度。

### 最佳文章类型
```yaml
效果排名: 待积累    # 如：故事型 > 观点型 > 干货型 > 清单型 > 热点型
各类型平均阅读量: 待积累
各类型平均分享率: 待积累
```

### 最佳赛道
```yaml
效果排名: 待积累    # 如：情感关系 > 职场成长 > 个人成长
各赛道平均阅读量: 待积累
涨粉最快赛道: 待积累
```

### 最佳开头方式
```yaml
效果排名: 待积累    # 如：场景代入 > 痛点共鸣 > 反常识
各方式平均完读率: 待积累
```

### 最佳发布时间
```yaml
最佳时段: 待积累    # 如：周二/四/六 20:00-21:00

```

### 最佳字数区间
```yaml
效果最好区间: 待积累    # 如：1800-2200字

```

## 三、关联分析规则

> 积累数据后自动执行以下分析。

| 积累条件 | 触发分析 |
|---------|---------|
| 5 篇同类型文章 | 该类型的效果趋势 + 最佳写法模式 |
| 10 篇任意文章 | 全面效果排名 + 最佳赛道/类型/开头/时间 |
| 20 篇任意文章 | 深度关联分析（哪些写作策略组合效果最好） |
| 每新增 5 篇 | 更新所有排名和模式 |
| 连续 3 篇效果差 | 预警 + 策略调整建议 |

## 四、效果基准线

> 根据积累数据自动计算，用于判断每篇文章效果好坏。

```yaml
平均阅读量: 待积累
平均分享率: 待积累    # 分享数/阅读量
平均互动率: 待积累    # (点赞+评论+分享)/阅读量
平均涨粉率: 待积累    # 涨粉数/阅读量
爆款阈值: 待积累      # 阅读量 ≥ 平均值 × 2
扑街阈值: 待积累      # 阅读量 ≤ 平均值 × 0.3
```
