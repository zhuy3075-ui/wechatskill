## 五、内容搭配原则

> 避免连续发同类型文章，保持内容多样性。

```
推荐搭配（每周 3 篇）：
- 模式 A：干货 + 故事 + 观点（最均衡）
- 模式 B：干货 + 清单 + 故事（偏实用）
- 模式 C：热点 + 观点 + 干货（偏时效）
- 模式 D：故事 + 观点 + 清单（偏情感）

避免：
- 连续 2 篇以上同类型
- 连续 3 篇以上同赛道
- 一周内没有任何故事型（故事型涨粉效果最好）
```

## 六、排期与选题库联动

> 从 topics.md 选题库中挑选待写选题填入排期。

操作流程：
1. 查看选题库中"待写"状态的选题，按爆款潜力排序
2. 根据本周内容搭配策略，挑选不同类型/赛道的选题
3. 填入本周排期表
4. 写完后更新选题库状态为"已写"
5. 发布后录入效果数据到 performance.md
