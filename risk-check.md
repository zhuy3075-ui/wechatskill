# 公众号文章风险检测清单

> 依据《微信公众平台运营规范》和《微信公众平台推荐运营规范》官方文件，对文章进行发布前全面风险检测。
> **规范来源**：
> - 运营规范：https://mp.weixin.qq.com/mp/opshowpage?action=newoplaw
> - 推荐规范：https://mp.weixin.qq.com/cgi-bin/announce?action=getannouncement&key=11697600328G0Tbo
>
> 每篇文章发布前必须逐项检查，确保零违规风险。

---

## 一、内容安全红线（一票否决，触发即永封）

以下任何一项触发，文章**必须放弃发布**。据运营规范，此类违规一经发现**永久封号**。

### 1.1 违反宪法和法律法规
- [ ] 不违反宪法确定的基本原则
- [ ] 不危害国家安全，不泄露国家秘密
- [ ] 不颠覆国家政权，不破坏国家统一
- [ ] 不损害国家荣誉和利益
- [ ] 不煽动民族仇恨、民族歧视，不破坏民族团结
- [ ] 不破坏国家宗教政策，不宣扬邪教和封建迷信
- [ ] 不散布谣言，不扰乱社会秩序，不破坏社会稳定

### 1.2 色情低俗
- [ ] 不散布淫秽、色情内容
- [ ] 不涉及招嫖、寻找一夜情等信息
- [ ] 不使用低俗配图引诱用户点击
- [ ] 不以低俗描述讲解或低俗图片视频展示成人用品
- [ ] 不发布擦边内容、展示低级趣味

### 1.3 暴力恐怖
- [ ] 不散播人或动物被杀、致残以及枪击、刺伤、拷打等受伤情形的真实画面
- [ ] 不传播恐怖主义或极端思想
- [ ] 不提供危险物品的制作方法
- [ ] 不展示死亡现场、器官伤害等令人观感不适的内容
- [ ] 不发布恐怖灵异内容

### 1.4 违法犯罪
- [ ] 不涉及赌博、毒品等违法内容
- [ ] 不发布替人复仇、收账等具有黑社会性质的信息
- [ ] 不教唆或引导违法行为
- [ ] 不涉及非法集资、传销（通过发展人员形成上下线关系，依据下线人数计算报酬）
- [ ] 不涉及网络借贷/信用卡代还、外汇交易、互联网黄金投资等需特许资质业务
- [ ] 不涉及虚拟货币发行、交易与融资，不为虚拟货币交易提供信息中介

## 二、不实信息检查（据运营规范+推荐规范）

微信明确规定：造谣源头账号根据谣言恶意程度、传播度、历史违规，给予**短封甚至永封**。

### 2.1 事实核查
- [ ] 不编造名人、明星、专家得绝症、去世等容易查证有违事实的内容
- [ ] 不编造、故意传播涉及食品安全、健康养生等日常生活方面的不实消息
- [ ] 不编造科学知识方面的不实消息
- [ ] 不偷换概念、断章取义，歪曲事件事实
- [ ] 不编造未经证实的信息
- [ ] 不利用 AIGC 制作虚假信息
- [ ] 文中数据有可靠来源，案例真实或已标注「虚构/化名」
- [ ] 引用的研究/报告真实存在

### 2.2 信息来源
- [ ] 发布时事新闻**准确标注信息来源**（运营规范明确要求）
- [ ] 不假借官方名义煽动获取流量（非官方通知但标题假借官方名义）
- [ ] 引述数据标明出处

### 2.3 联网核查与多信源验证

> 核查不是每篇文章都要做，也不是查一个来源就完事。
> 原则：**克制触发、聚焦时政、报告先行、用户决策**。

#### 触发条件

| 触发场景 | 核查级别 | 触发方式 |
|---------|---------|---------|
| 用户明确说"帮我核查""这个是真的吗""验证一下" | 完整核查 | 必须执行 |
| 文章涉及时事政治、政策法规、政府动态 | 事件+时效核查 | 自动触发 |
| 文章涉及重大社会事件/突发新闻（有明确时间线） | 事件核查 | 自动触发 |
| 文章引用了具体数据/统计（如"85%的人……"） | 数据核查 | 仅提醒，用户确认后执行 |
| 文章引用名人名言/专家观点 | 引用核查 | 仅提醒，用户确认后执行 |
| 文章涉及医疗、法律、金融等专业领域的事实性陈述 | 专业核查 | 仅提醒，用户确认后执行 |
| 质检阶段发现可疑数据或事件 | 定向核查 | 仅提醒，用户确认后执行 |

**触发控制规则：**
- 非时政类文章（如情感、职场感悟、生活方式）默认不触发核查，除非用户要求
- 同一篇文章最多自动触发 1 次核查（避免反复打断创作流程）
- "仅提醒"类场景：在质检报告中标注「建议核查」，由用户决定是否执行
- 提醒格式：`💡 发现 N 处可核查内容（数据引用 x 处、名人名言 x 处），是否需要联网核查？`

#### 核查流程

```
需要核查的内容
  │
  ├─ 第一步：明确核查对象
  │   └─ 提取文章中所有需要核查的事实性陈述（数据、事件、引用、时间线）
  │
  ├─ 第二步：多信源搜索（至少 2-3 个独立来源）
  │   ├─ WebSearch 搜索关键词（多角度搜索，不只搜一次）
  │   │   ├─ 搜索原始表述
  │   │   ├─ 搜索关键实体 + 时间
  │   │   └─ 搜索反面/质疑观点（排除确认偏误）
  │   ├─ 如需深入 → WebFetch 访问具体来源页面获取详情
  │   └─ 如来源为微信文章 → Playwright MCP 抓取
  │
  ├─ 第三步：交叉验证
  │   ├─ 比对多个来源的信息是否一致
  │   ├─ 评估来源可信度（官方 > 权威媒体 > 行业报告 > 自媒体）
  │   ├─ 检查信息时效性（是否过时）
  │   └─ 识别信息链条（是否所有来源都指向同一个原始出处 → 实际只有一个来源）
  │
  └─ 第四步：输出核查报告
```

#### 来源可信度分级

| 级别 | 来源类型 | 可信度 |
|------|---------|-------|
| A | 政府官网、官方统计局、央行等 | 高度可信 |
| A | 权威学术期刊、Nature/Science 等 | 高度可信 |
| B | 主流媒体（新华社、人民日报、央视、财新等） | 可信 |
| B | 行业权威报告（麦肯锡、艾瑞、QuestMobile 等） | 可信 |
| C | 知名自媒体、行业 KOL | 需交叉验证 |
| C | 百度百科、维基百科 | 需交叉验证（可作为线索，不作为唯一来源） |
| D | 不明来源、营销号、无署名文章 | 不可信（不可作为核查依据） |

#### 核查报告输出格式

```
🔍 事实核查报告

═══ 核查对象 ═══
「[需要核查的原文表述]」

═══ 核查结果 ═══
✅ 属实 / ⚠️ 部分属实 / ❌ 不实 / ❓ 无法确认

═══ 信源验证 ═══
来源 1：[来源名称]（可信度：A/B/C）
  - 内容：[该来源的相关信息]
  - 链接：[URL]

来源 2：[来源名称]（可信度：A/B/C）
  - 内容：[该来源的相关信息]
  - 链接：[URL]

来源 3：[来源名称]（可信度：A/B/C）
  - 内容：[该来源的相关信息]
  - 链接：[URL]

═══ 交叉验证结论 ═══
- 多源一致性：[一致 / 部分矛盾 / 严重矛盾]
- 信息时效性：[最新 / 可能过时 / 已过时]
- 原始出处追溯：[已找到原始出处 / 所有来源指向同一出处 / 无法追溯]

═══ 处理建议（仅供参考，由您决定是否采纳） ═══
✅ 属实 → 可以使用，建议标注来源
⚠️ 部分属实 → 可考虑修正表述，修改建议：[建议]
❌ 不实 → 建议删除或替换，替代方案：[建议]
❓ 无法确认 → 建议删除，或改为模糊表述（如"有观点认为……"）

👉 请告诉我：需要按建议修改 / 保持原文 / 部分采纳（请指定）
```

#### 批量核查格式

文章中有多个需要核查的点时，汇总输出：

```
🔍 批量核查报告（共 N 项）

1. 「85%的年轻人有焦虑问题」 → ⚠️ 部分属实
   实际数据：中国科学院心理研究所 2023 报告显示约 67%，非 85%
   建议：修正为"超过六成年轻人"并标注来源

2. 「某公司 2024 年裁员 30%」 → ✅ 属实
   来源：财新网 2024-03-15 报道 + 公司财报确认
   建议：可以使用，标注来源

3. 「爱因斯坦说过'想象力比知识更重要'」 → ⚠️ 部分属实
   实际：出自 1929 年 Saturday Evening Post 采访，但常被断章取义
   建议：补充完整语境或改为"爱因斯坦曾在采访中表示……"

总结：N 项核查，X 项属实，Y 项需修正，Z 项不实

👉 请选择处理方式：
  A. 全部按建议修改
  B. 仅修改不实/部分属实的内容
  C. 我逐条确认
  D. 暂不修改，保持原文
```

## 三、标题风险检查（据运营规范+推荐规范）

微信明确禁止以下标题行为，违规将导致**文章不被推荐、删文、封禁**：

- [ ] 标题不使用危害人身安全、恐吓侮辱、惊悚、极端内容
- [ ] 标题不使用命令式语气强迫用户
- [ ] 标题不无依据夸大事件严重程度、紧急程度、受影响面
- [ ] 标题不故意隐藏关键信息，不无中生有部分信息
- [ ] 标题不假借官方名义（非官方通知但标题假借官方名义煽动获取流量）
- [ ] 标题与内容一致，不存在标题党（内容无法兑现标题承诺）
- [ ] 标题不使用「震惊」「重磅」「突发」等浮夸描述
- [ ] 标题不暗示色情或低俗内容
- [ ] 标题不涉及名人/明星的未经证实信息
- [ ] 标题信息完整，不影响阅读理解

## 四、诱导行为检查（据运营规范，违规处罚：禁转发/删文/删粉/封号）

微信对诱导行为处罚严厉，根据违规程度采取**文章禁转发、删文、删粉、封禁相关能力，甚至封号**。

### 4.1 诱导分享（严禁）
- [ ] 不强制用户分享（分享后才能继续下一步操作）
- [ ] 不利诱用户分享（邀请好友助力、集赞，可获得抽奖机会）
- [ ] 不胁迫用户分享（不使用「不转不是中国人」「必转」等夸张言语）
- [ ] 不使用「转发后好运」等诱导话术

### 4.2 诱导关注（严禁）
- [ ] 不强制或暗示用户关注才能获得好处或进行下一步操作
- [ ] 不利诱用户关注（关注可获得红包/抽奖机会）
- [ ] 不使用「点赞的人都会XXX」等诱导点赞/在看话术
- [ ] 不使用「回复XXX查看」等虚假互动话术

### 4.3 违规导流
- [ ] 文章主要目的不是诱导用户点击跳转
- [ ] 不通过封建迷信、违规擦边标题吸引用户
- [ ] 不直接放置其他平台的链接进行导流

## 五、版权与原创检查（据运营规范）

违规将被**取消原创标识**，并根据违规情况对账号予以相应处理。

### 5.1 版权合规
- [ ] 图片有版权（使用无版权图库或自有图片）
- [ ] 引用他人文字标注来源，引用篇幅在合理范围内
- [ ] 不存在洗稿嫌疑（大量复述仅作轻加工、重组拼接）
- [ ] 音视频素材有使用授权
- [ ] 不带有其他平台 logo 或水印
- [ ] 不混淆大 IP 品牌身份

### 5.2 原创规范（以下情形不得标注原创）
- [ ] 未经授权不合理引用或大篇幅整合网络信息 → 不标原创
- [ ] 营销宣传内容 → 不标原创
- [ ] 非独家授权内容 → 不标原创
- [ ] 无实际意义、无指导性、文字过于宽泛的内容 → 不标原创
- [ ] 整合剪辑的视频内容 → 不标原创

## 六、广告与营销检查（据运营规范）

### 6.1 广告合规
- [ ] 第三方营销**依法标明「广告」字样**（运营规范明确要求）
- [ ] 医疗、药品广告**经过广告审查机关审查**
- [ ] 广告内容合法合规，不存在虚假宣传

### 6.2 过度营销禁止项
- [ ] 不使用夸大的描述推销丰胸、减肥、增高等保健品
- [ ] 推销的产品不明显低于官方价格
- [ ] 非品牌官方发布的产品不冒充官方
- [ ] 非医疗用品不明示或暗示产品具备医疗功效
- [ ] 不使用「最」「第一」「唯一」等绝对化广告用语

## 七、特殊领域合规检查

### 7.1 医疗健康类（据推荐规范：未证实偏方、夸大药效、无资质行医均违规）
- [ ] 不提供具体的诊断建议
- [ ] 不推荐具体的处方药
- [ ] 不宣传未经批准的疗法、未证实的偏方
- [ ] 不夸大药物效果
- [ ] 不进行无资质行医宣传
- [ ] 附有「本文仅供健康科普参考，不能替代专业医疗建议」免责声明
- [ ] 不使用：治愈、根治、特效药、秘方、祖传、速效等词汇

### 7.2 金融理财类（据运营规范：需特许资质类业务严禁）
- [ ] 不承诺具体收益
- [ ] 附有投资风险提示
- [ ] 不涉及网络借贷/信用卡代还/外汇交易/互联网黄金投资
- [ ] 不涉及虚拟货币/数字货币的交易引导
- [ ] 不使用：保本、稳赚、高收益、零风险、翻倍等词汇

### 7.3 法律相关类
- [ ] 附有「本文不构成法律意见，具体法律问题请咨询专业律师」免责声明
- [ ] 法律条文引用准确且为最新版本
- [ ] 案例分析客观中立

### 7.4 教育培训类
- [ ] 不使用虚假的成功案例
- [ ] 不承诺学习效果（保过、包会、包就业）
- [ ] 不使用：保过、包就业、提分等承诺性词汇

### 7.5 涉及未成年人（据运营规范）
- [ ] 不公开未成年人个人信息
- [ ] 不使用未成年人照片（未经监护人同意）
- [ ] 内容不对未成年人造成不良引导
- [ ] 不诱导未成年人进行明星应援、参加粉丝团或后援会
- [ ] 不出现描绘暴力或虐待儿童等内容

## 八、推荐规范检查（据推荐运营规范，影响文章曝光量）

以下内容**不会被推荐或被取消推荐**，直接影响文章阅读量：

### 8.1 低创作度内容（会被降权/不推荐）
- [ ] 不短期内高频发布结构、框架、核心观点完全一致的同质化内容
- [ ] 不大量复述他人内容仅作轻加工（疑似搬运）
- [ ] 不进行洗稿重组、机械拼接
- [ ] 不堆砌套话缺乏实质内容
- [ ] 不存在图文失配
- [ ] 不虚构未验证的数据

### 8.2 AIGC 内容规范
- [ ] 不直接使用 AI 生成内容占比过高的文章
- [ ] AI 辅助创作的内容不包含事实错误
- [ ] 不使用 AI 批量灌水式生产内容
- [ ] 必要时声明 AI 辅助创作

### 8.3 违背公序良俗（不推荐）
- [ ] 不恶搞灾难事件
- [ ] 不宣扬流量至上、拜金主义
- [ ] 不煽动性别对立
- [ ] 不展示低级趣味、擦边内容

### 8.4 过时信息（不推荐）
- [ ] 不发布时效已失效的政策法规信息
- [ ] 不无新观点地二次传播旧内容
- [ ] 不扭曲历史背景

### 8.5 封面图规范（不合规将不被推荐）
- [ ] 封面图中人物完整，不存在明显裁切
- [ ] 封面图画质清晰，不模糊
- [ ] 不使用鬼怪正脸等直观惊悚形象
- [ ] 封面图与内容相符，不存在误导
- [ ] 封面图不包含血腥、暴力、色情内容
- [ ] 封面图不包含未经授权的人物肖像或品牌 Logo

## 九、鼓励的内容方向（据推荐规范，有利于获得推荐）

创作时优先选择以下方向，有助于获得平台推荐：

- **传递知识、分享经验的原创内容**
- **具有启发性、教育性或娱乐性质的内容**
- **专业人士发布的权威真实信息**
- **提供良好阅读体验和精美排版的内容**
- **提供信息增量或情感价值的内容**

## 十三、质检执行规范

> 质检不是可选步骤，是每篇文章的必经流程。
> 质检结果必须显式输出给用户，格式详见 `self-evolution.md` 第零节 0.5。


### 字数检查规则（用户指定字数时）

> **当用户明确规定了字数要求时，字数检查为高优先级质检项，必须严格执行。**

**检查标准：**
- 允许误差范围：**上下浮动不超过 5%**
- 例：用户要求 2000 字 → 合格范围 1900~2100 字
- 例：用户要求 3000 字 → 合格范围 2850~3150 字

**检查时机：**
- 初稿完成后第一时间检查字数，字数不达标则优先调整
- 每次修改后重新统计字数并校验

**不达标处理：**
```
字数偏少（低于下限）→ 自动扩充内容（加案例、加论述、加金句），直到达标
字数偏多（超过上限）→ 自动精简（删冗余、合并段落、缩短过渡），直到达标
调整后必须重新校验字数，确认在 ≤5% 误差范围内
```

**质检报告字数展示格式：**
```
═══ 字数检查 ═══
目标字数：2000 字（用户指定）
实际字数：1985 字
偏差：-0.75%（允许范围 ±5%）
结果：✅ 达标
```
