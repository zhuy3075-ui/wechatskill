# wechat-writer 用户操作手册

> 适合直接发给最终用户。按步骤操作即可完成写作、范文学习与自学习进化。

## 1. 激活

你可以用两种方式开始：

1. 自然语言（推荐）  
`写一篇观点文章，主题是AI对教育的影响`
2. 命令式  
`/wechat-writer 观点 AI对教育的影响`

## 2. 写作流程（一步步）

1. 提供主题或素材  
`把这段素材写成故事型公众号文章`
2. 选择风格（可选）  
`用理性派风格写`
3. 先看大纲再写全文（推荐）  
`先给我大纲，我确认后再写`
4. 选择输出格式  
- `md`：Markdown  
- `json`：结构化  
- `both`：两者都要  
- `wechat`：公众号可直贴版  
示例：`输出格式用 wechat`

## 3. 范文学习功能（重点）

### 3.1 如何触发

1. 提供范文内容（文本/链接/文件）
2. 必须补充作者名或账号名

示例：
`学习这篇范文，作者是 XXX`

### 3.2 系统会做什么

1. 识别文章类型与结构特征
2. 拆解语气、文风、节奏、金句、标题技巧
3. 抽取素材到 `memory/materials.md`
4. 生成风格档案到 `styles/[作者名].md`
5. 冲突检测：判断是否影响当前人格

### 3.3 学完如何调用

`用XXX风格写这篇`

## 4. 自学习进化功能（重点）

### 4.1 会自动学习什么

1. 你最终选的标题与改动偏好
2. 文章里的有效金句与结构
3. 素材复用情况（M-XXX）
4. 发布效果数据（阅读、互动等）

### 4.2 会写入哪里

1. `memory/titles.md`：标题偏好和高效模式
2. `memory/golden-sentences.md`：金句沉淀
3. `memory/materials.md`：案例/数据/标题技巧等素材
4. `memory/performance.md`：效果复盘
5. `personality.md`：人格慢速微调（受锁定与冲突规则约束）

### 4.3 你可以主动触发

1. `记住这个偏好`
2. `阶段复盘 最近10篇`
3. `导出优化方案`
4. `锁定风格` / `解锁风格`

## 5. 偏好确认（正文被优化后）

当系统对正文做形式优化时，会给你一组可确认偏好：

1. 短句密度（克制/标准/宽松）
2. 破折号偏好（低/中/高）
3. 俚语方言偏好（按场景启用/少量固定/禁用）

你只需回复：

1. `确认全部`
2. 或 `按编号修改`（例如：`1改为克制，3改为按场景启用`）

## 6. 常见问题

1. 为什么没进入范文学习？  
最常见原因：没有提供作者名/账号名。
2. 为什么风格和我不一致？  
检查是否指定了 `--style`，以及是否锁定了风格。
3. 为什么排版粘贴到公众号不理想？  
用 `wechat` 模式输出。
4. 如何减少“AI味”？  
让系统先过质量闸门，再输出最终稿。
