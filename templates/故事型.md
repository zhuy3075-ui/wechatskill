# 故事型文章模板

> 适用于：个人经历、案例分享、人物故事、成长复盘类文章

## 文章结构

```
标题：[人物/身份] + [转折/冲突] + [结果/感悟]
示例：「从月薪 3000 到年入百万，她只做对了一件事」

---

## 开头（100-200 字）

[场景化开头] —— 用画面感最强的一个瞬间切入
[制造悬念] —— 让读者想知道后来怎么了
[暗示主题] —— 隐约点出这个故事要讲什么

---

## 一、[起点 / 背景交代]

[人物背景]：用 2-3 个细节快速建立人物形象
[初始状态]：故事开始时的处境
[目标/欲望]：主人公想要什么

## 二、[冲突 / 困难]

[遇到什么困难]：具体的障碍和挫折
[内心挣扎]：心理描写增强代入感
[关键细节]：用一个细节让场景鲜活起来

## 三、[转折 / 突破]

[转折点]：什么事情改变了局面
[行动]：做了什么不同的事
[过程]：怎么一步步走出来的

## 四、[结果 / 感悟]

[结果]：最终怎么样了
[感悟提炼]：这个故事告诉我们什么
[与读者的连接]：让读者看到自己的影子

> [金句：从故事中提炼的人生感悟]

---

## 结尾（100-150 字）

[回到当下]：故事主人公现在怎样
[点明主题]：用一句话概括这个故事的意义
[情感共鸣]：让读者产生情感连接
[互动引导]：邀请读者分享类似经历

```

## 爆文公式

实战中验证有效的故事型爆文公式：

```
标题（开幕雷击：金钱/异常/悬念等要素组合）
+ 主人公介绍（我是谁，多少岁，来自哪里，做什么的）
+ 相关人物（与主人公的关系，怎么认识的）
+ 故事主干（想做什么 → 发生了什么 → 遇到什么问题）
  包含：目标或梦想 + 挫折或阻碍或意外 + 反转或惊喜
+ 结局（对主人公的影响 + 怎么做的 + 结果是什么）
```

**爆款故事必备条件**：
1. 好的标题（包含开幕雷击要素）
2. 原创（不是洗稿）
3. 有信息增量（不是旧闻）
4. 不重复新闻
5. 不传播负能量
6. 结尾尽量反转——意料之外、情理之中

**关键技巧**：
- 一般用第一人称写，方便读者代入
- 与主题无关的内容略写，能推动故事的情节详写
- 开头 5 句话交代清楚：人物、职业、关系、引出悬念
- 故事要流畅，逻辑清晰，转折不生硬

## 写作要点

1. **细节是灵魂**：一个好细节胜过十句概述
2. **用场景而非概念**：「她在深夜的出租屋里哭了」比「她很难过」好 100 倍
3. **有起有伏**：故事要有低谷和高潮
4. **对话推动叙事**：适当使用对话让场景活起来
5. **感悟要自然**：从故事中长出来，不要强行贴标签
6. **真实最有力量**：不要虚构关键事实
7. **多写行为、动作、语言**：少写内心想法，让读者自己感受
8. **注意转折词**：「但是」「为何」「不过」「然而」是文章的呼吸节奏

## 开头 Hook 模板

### 模板 A：画面切入
```
[时间]，[地点]，[人物在做什么]。

[一个有画面感的细节]。

[悬念句：这一切是怎么开始的 / 谁能想到]。
```

### 模板 B：结果先行
```
[惊人的结果/现状]。

但 [时间] 前，[完全相反的状态]。

这中间发生了什么？
```

### 模板 C：对话开场
```
「[一句有冲击力的对话]。」

[说这话的人是谁]，[当时的场景]。

[后来怎样了]。
```

## 结尾模板

### 模板 A：回到现在
```
现在的 [主人公]，[现状描写]。

偶尔 [回忆的细节]，[他/她] 会 [一个小动作/表情]。

有些路，走过才知道值不值。但走过的每一步，都算数。
```

### 模板 B：给读者的话
```
讲完这个故事，我想对你说——

如果你现在也在 [类似的困境]，别急。

[鼓励/感悟]。

你经历过类似的时刻吗？评论区等你 👇
```

## 故事写作核心技巧

### 五感描写
调动视觉、听觉、触觉、嗅觉、味觉，让读者身临其境：
- 视觉：「办公室的灯一盏盏灭了，只剩她工位上那一盏」
- 听觉：「手机响了，是那个存了三年没拨出去的号码」
- 触觉：「合同上的字很烫，签名的手在抖」

### 时间线处理
- **顺叙**：适合成长型故事
- **倒叙**：适合结果惊人的故事（先说结果，再讲过程）
- **插叙**：在关键时刻插入回忆，增加深度
