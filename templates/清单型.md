# 清单型文章模板

> 适用于：盘点推荐、工具合集、书单影单、方法汇总类文章

## 文章结构

```
标题：[数字] + [清单内容] + [价值承诺]
示例：「私藏 3 年的 10 个效率工具，每一个都好用到哭」

---

## 开头（100-150 字）

[为什么做这个清单] —— 说明筛选标准和信息来源
[清单价值] —— 告诉读者看完能获得什么
[使用建议] —— 建议怎么使用这个清单（收藏/挑选适合自己的）

---

## 1. [清单项名称]

**一句话推荐**：[用一句话说清楚为什么推荐]

[详细介绍]：2-4 句话展开说明
[亮点/特色]：最突出的 1-2 个优势
[适合谁]：什么人/场景最适合（可选）

## 2. [清单项名称]

（同上结构）

## 3. [清单项名称]

（同上结构）

……

---

## 结尾（100-150 字）

[使用建议]：不要贪多，挑 1-2 个开始用
[互动引导]：邀请读者补充他们的推荐
[收藏提醒]：这种文章适合提醒收藏

```

## 爆文公式

实战中验证有效的清单型爆文公式：

```
标题（数字 + 盘点/推荐/合集 + 开幕雷击要素）
+ 开头（为什么做这个清单 + 筛选标准 + 价值承诺）
+ N 个条目（名称 → 推荐理由 → 亮点 → 适用场景）
+ 结尾（使用建议 + 互动 + 收藏引导）
```

**关键技巧**：
- 条目间结构高度相似，形成阅读节奏
- 每项都要有评价，不是简单罗列
- 条目数量是标题的核心卖点，用数字吸引眼球

## 写作要点

1. **每项都要有评价**：不能只是罗列，要说清楚好在哪里
2. **排序有逻辑**：按推荐度/使用频率/难度等排序
3. **筛选比罗列更有价值**：10 个精选好过 50 个堆砌
4. **个人体验加分**：加入自己的使用感受更真实
5. **分类整理**：超过 7 项建议分组归类

## 开头 Hook 模板

### 模板 A：筛选价值
```
为了这份清单，我 [花了多长时间/测试了多少个/翻了多少资料]。

最终从 [大数字] 个里面，精选出这 [小数字] 个，每一个都经过亲测。

建议先收藏，用到的时候随时翻出来。
```

### 模板 B：痛点引入
```
每次 [遇到某个场景]，是不是都要纠结半天？

今天这份清单帮你省去选择的时间。[N] 个 [主题]，闭眼入都不会出错。
```

## 清单项写作模板

### 简洁版（适合 10+ 项的长清单）
```
**[序号]. [名称]**
[一句话推荐理由]。[一个具体亮点或使用场景]。
```

### 详细版（适合 5-7 项的精选清单）
```
## [序号]. [名称]

**推荐指数**：⭐⭐⭐⭐⭐

[2-3 句详细介绍]

**最大亮点**：[一句话说清楚]
**适合人群**：[谁最该用/看/读]
**个人体验**：[你自己用了之后的感受]
```

## 结尾模板

### 模板 A：互动收藏
```
以上就是我精选的 [N] 个 [主题]，每一个都是亲测好用的。

不要贪多——先从列表里挑一个，今天就用起来。

你还有什么好用的 [主题] 想推荐？评论区一起交流，说不定你的推荐比我的更好 👇

觉得有用的话，记得收藏，免得下次找不到了。
```
