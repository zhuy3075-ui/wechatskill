# 热点型文章模板

> 适用于：追热点、时事评论、事件解读、趋势分析类文章

## 文章结构

```
标题：[热点事件关键词] + [独特角度/深层解读]
示例：「全网都在骂 XX，但真正的问题不在这里」

---

## 开头（100-200 字）

[热点事件速览] —— 用 2-3 句话说清楚发生了什么
[大众反应] —— 网上主流的讨论方向
[你的切入角度] —— 你准备从什么角度来聊

---

## 一、[事件回顾 / 发生了什么]

[事实梳理]：客观、准确地还原事件经过
[关键细节]：大家可能忽略的重要信息
[不同声音]：各方的反应和立场

## 二、[深度分析 / 为什么会这样]

[表面原因]：大家都看到的原因
[深层原因]：你发现的更深层的逻辑
[类比/历史]：类似的事件有哪些，规律是什么

> [金句：一句话概括这件事的本质]

## 三、[启示 / 对我们的影响]

[对普通人意味着什么]：跟读者建立关联
[我们该怎么看/怎么做]：给出建设性的思考
[趋势判断]：未来可能怎么发展

---

## 结尾（100-150 字）

[克制表态]：表明态度但不极端
[理性呼吁]：呼吁理性看待
[开放讨论]：邀请读者说出自己的看法

```

## 爆文公式

实战中验证有效的热点文公式（参考龙骨龙鳞法，详见 rules/style-guide.md）：

### 简单版
```
标题（开幕雷击 + 50 字以内）
+ 开头（100 字左右，可以是自己的领悟或直接进入事件）
+ 事件陈述（500 字左右，配相关图片）= 龙骨
+ 结论（300 字左右，论点 + 激发情绪）= 龙鳞
```

### 深度版
```
标题 + 开头 + 事件陈述 + 对比论证（300 字，找类似事件作比对）+ 结论
```

### 不同热点类型的选题思路

| 类型 | 选题来源优先级 | 目标读者 |
|------|--------------|---------|
| 指点江山类（社会不公） | 知乎自然爆款 > 知乎热榜 > 微博热榜 | 中老年男性 |
| 娱乐热点类 | 饭圈小道消息 > 微博热榜 > 知乎热榜 | 中青年女性 |
| 养生热点类 | 节气 ≥ 微博等平台健康类选题 | 中老年（不分性别） |

**注意**：养生类只能往古代习俗上靠，不要给任何医学建议，否则严重违规。

## 写作要点

1. **速度很重要**：热点文章越快越好，但不能牺牲准确性
2. **事实先行**：先把事情说清楚，再发表观点
3. **独特角度是关键**：人云亦云不如不写，要有自己的洞察
4. **不消费负面情绪**：分析问题而非煽动情绪
5. **标注信息来源**：提到的数据和事实要有出处
6. **留有余地**：热点事件往往会反转，不要把话说太满
7. **龙骨要短**：事件太复杂就只讲一个侧面，不追求全面
8. **龙鳞要有力**：情绪上来就写出来，用它争取和你共鸣的读者

## 写作禁忌

- ❌ 传播未经证实的信息
- ❌ 对当事人进行人身攻击
- ❌ 过度消费他人痛苦
- ❌ 以偏概全，用个案推导出普遍结论
- ❌ 绝对化表达：「一定」「肯定」「必然」
- ❌ 蹭与自己内容领域完全无关的热点

## 开头 Hook 模板

### 模板 A：速报式
```
[事件]，刷屏了。

[关键数据/反应]。

但在这场全民讨论中，有一个角度被忽略了。
```

### 模板 B：反共识式
```
关于 [热点事件]，全网几乎是一边倒的 [主流观点]。

但我想泼一盆冷水——事情可能没那么简单。
```

### 模板 C：提问式
```
[热点事件] 之后，每个人都在问：[大家都在问的问题]？

但我觉得，真正该问的问题是：[你认为该问的问题]？
```

## 结尾模板

### 模板 A：理性收尾
```
热点终会过去，但这件事给我们的启示不应该被遗忘：

[一句话总结启示]。

最后，在事情完全明朗之前，少一些情绪，多一些思考。

你怎么看这件事？评论区聊聊 👇
```

### 模板 B：行动导向
```
与其在网上争论，不如想想我们能做什么。

[一个具体的小建议/行动]。

比起当键盘侠，这才是更有意义的事。
```

## 热点文章时效性建议

| 时间窗口 | 内容策略 |
|---------|---------|
| 0-2 小时 | 快速信息整合，事实梳理 |
| 2-12 小时 | 深度分析，独特视角 |
| 12-48 小时 | 复盘总结，延伸思考 |
| 48 小时后 | 除非有新进展，否则不建议再追 |
