# 干货型文章模板

> 适用于：教程、知识分享、方法论、技能提升类文章

## 文章结构

```
标题：[数字] + [具体收益] + [目标群体/场景]
示例：「学会这 5 个方法，你的学习效率至少翻一倍」

---

## 开头（150-200 字）

[痛点场景] —— 用 1-2 句话描述读者正在经历的困境
[数据/案例佐证] —— 说明这个问题的普遍性
[解决方案预告] —— 告诉读者这篇文章能解决什么

---

## 一、[第一个要点/方法/步骤]

[要点阐述]：用 2-3 句话说清楚是什么
[为什么有效]：解释原理或逻辑
[具体做法]：步骤化说明怎么做
[案例/效果]：用真实案例或数据说明效果

## 二、[第二个要点/方法/步骤]

（同上结构）

## 三、[第三个要点/方法/步骤]

（同上结构）

> 💡 [金句总结：一句话概括核心方法论]

---

## 结尾（100-150 字）

[总结回顾]：一句话概括全文核心
[行动号召]：给读者一个立刻能做的小行动
[互动引导]：提问引发评论

```

## 爆文公式

实战中验证有效的干货型爆文公式（以职场类为例）：

```
标题（含开幕雷击要素 + 方法/数字关键词）
+ 故事主干（用一个具体案例/截图引出主题，进行讨论分析）
+ 网友争议点（展示不同观点，对比解析，给出自己的观点）
+ 总结建议（可操作的行动指引）
```

**关键技巧**：
- 标题写得好 = 70% 的成功（进入流量池后内容差距反而缩小）
- 分析原因时学会讲故事，比枯燥的分析更吸引人
- 确定好选题后，将主题截图发布到文章顶部增加真实感
- 每个章节有可操作的行动指引，不要只讲道理

## 写作要点

1. **标题必须包含数字**：数字给人确定感和获得感
2. **每个方法都要可操作**：读者看完就能用，不要只讲道理
3. **案例比道理更有说服力**：先讲案例再总结方法
4. **由浅入深排列**：先讲最容易上手的，再讲进阶的
5. **适当使用清单/步骤**：降低理解成本
6. **学会讲故事**：在干货中穿插小故事，比纯分析更吸引人

## 开头 Hook 模板

### 模板 A：痛点共鸣
```
你是不是也有过这样的经历——[描述痛点场景]？

别担心，你不是一个人。[数据说明普遍性]。

今天分享 [N] 个方法，帮你彻底解决这个问题。
```

### 模板 B：反常识开场
```
大多数人以为 [常见认知]，但事实恰恰相反。

[反常识观点 + 数据/证据]。

我花了 [时间] 总结出 [N] 条经验，今天一次性分享给你。
```

### 模板 C：成果展示
```
[我/某人] 用了 [这个方法]，[具体成果]。

这不是天赋，而是方法。

今天把这套方法拆解给你，看完就能用。
```

## 结尾模板

### 模板 A：总结 + 行动
```
最后总结一下今天分享的 [N] 个方法：
1. ……
2. ……
3. ……

别光收藏，今天就试试第一个方法。哪怕只做 5 分钟，也比什么都不做强。

你觉得哪个方法最实用？评论区聊聊 👇
```

### 模板 B：升华 + 互动
```
[一句金句总结全文]

方法给你了，剩下的就看执行力了。

你在 [主题] 方面踩过什么坑？欢迎在评论区分享你的经验，我们一起交流。
```
