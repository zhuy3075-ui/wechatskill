# 微信公众号写作风格指南

## 一、整体风格原则

### 1. 对话感
- 像跟朋友聊天一样写作，不是写论文
- 多用「你」「我」，少用「笔者」「读者」
- 适当使用口语化表达，但不低俗

### 2. 信息密度
- 每句话都要有信息量，删除一切废话
- 一个段落只讲一个意思
- 重要信息前置，不要埋在段落中间

### 3. 节奏感
- 长短句交替使用
- 适当使用短句制造冲击力
- 关键处用独立成段的短句强调

### 5. 全局偏好 vs 风格偏好
- **全局偏好（所有风格通用）**：可读性、信息密度、合规底线、不过度碎句、不过度标点噪音。
- **风格偏好（各风格独立）**：语气强弱、口头禅、标点怪癖、情绪力度、俚语/方言浓度。
- 执行优先级：全局偏好 > 风格偏好。若风格偏好影响可读性，自动回退到全局规则。

### 4. 画面感
- 多用具体的场景描写，少用抽象概念
- 用数字说话：「3 天」比「几天」更有力
- 用对比制造落差：「月薪 3000 到 30000」

## 二、不同文章类型的语气

### 干货型
- **语气**：专业但不学术，权威但亲和
- **人称**：多用「我」分享经验，「你」给建议
- **特点**：条理清晰，步骤明确，可操作性强
- **禁忌**：不要说教，不要居高临下
- **示例语气**：
  > 很多人学编程都踩过这个坑——上来就啃大部头教材。其实更好的方式是：先做一个小项目，遇到问题再去查。我自己就是这么过来的。

### 观点型
- **语气**：犀利但理性，有态度但不偏激
- **人称**：「我」表明立场，适当用「我们」拉近距离
- **特点**：观点鲜明，论据充分，逻辑自洽
- **禁忌**：不要人身攻击，不要绝对化表达
- **示例语气**：
  > 「内卷」这个词被用烂了，但真正的问题从来不是竞争本身——而是大家都在同一条赛道上卷相同的东西。换个赛道，你会发现世界宽得很。

### 故事型
- **语气**：真诚、有温度、有细节
- **人称**：第一人称为主，营造沉浸感
- **特点**：有起伏、有转折、有情感共鸣
- **禁忌**：不要虚构关键事实，不要过度煽情
- **示例语气**：
  > 2019 年冬天，我坐在出租屋里，银行卡余额 237 块。手机亮了，是妈妈的微信：「吃饭了吗？」我盯着屏幕，打了三个字又删掉。最后回了一个字：「嗯。」

### 清单型
- **语气**：简洁、干脆、有节奏
- **人称**：灵活，可用「你」给推荐，用「我」分享体验
- **特点**：每个条目都要有亮点描述，不能只是罗列
- **禁忌**：不要流水账式列举，每项要有评价/理由
- **示例语气**：
  > **3. 《被讨厌的勇气》**
  > 如果你总是在意别人的看法，这本书会像一记耳光把你打醒。它不是鸡汤，而是一套完整的「课题分离」方法论。读完你会发现——别人怎么想，真的不关你的事。

### 热点型
- **语气**：快速、有洞察、不跟风
- **人称**：灵活，重在分析而非个人表达
- **特点**：快速切入热点，提供独特视角，不人云亦云
- **禁忌**：不要传播未经证实的信息，不要消费负面情绪
- **示例语气**：
  > 全网都在讨论 XX 事件，但很少有人注意到一个关键细节——……这个细节背后，其实藏着一个更深层的问题。

## 三、用词规范

## 人味增强器（优先执行）

在不改变事实、观点和结论的前提下，优先做下面 5 件事：

1. 把抽象判断落到微场景：时间 + 动作 + 感受（至少两项）。
2. 每个核心段加入“说人话”的短句，避免全段都像报告。
3. 增加作者判断句：明确你赞成/反对/担心什么。
4. 让句长有起伏：短句打断 + 中句解释 + 长句展开。
5. 删除空泛正确话：能删就删，留有信息密度的句子。

## 反模板化控制（降低AI风格风险）

1. 段落不要“工整到平均”：长短段交错出现。
2. 小标题避免同构：不要连续使用同一语法骨架。
3. 连接词克制：`首先/其次/最后` 只在必要时使用，且不连续成链。
4. 每个小节至少保留一个“人类痕迹点”：
   - 临场判断句（如“我更倾向于…”）
   - 回头补充句（如“这里要补一句…”）
   - 微场景（时间+动作）
5. 允许轻微不对称：
   - 个别段落只有一句话
   - 个别段落含转折打断（破折号/反问）
6. 严禁伪人味：
   - 生硬堆叠口语词
   - 无信息量的情绪句

### 推荐用词
| 避免使用 | 推荐替换 |
|---------|---------|
| 笔者 | 我 |
| 读者朋友们 | 你 |
| 众所周知 | （直接省略，进入正题） |
| 接下来 | （用小标题替代过渡） |
| 综上所述 | 说到底 / 总结一下 |
| 不可否认 | 确实 / 没错 |
| 究其原因 | 为什么？因为…… |
| 值得一提的是 | （直接说，不要铺垫） |

### 禁用表达
- 「关注我，带你了解更多」（太生硬）
- 「不转不是中国人」类诱导分享
- 「震惊！」「重磅！」等标题党词汇
- 「小编」自称（不专业）
- 过度使用网络流行语（容易过时）

### 标点符号
- 省略号用「……」（6个点），不用「。。。」
- 破折号用「——」（两个），不用「--」
- 引用他人观点用「」（直角引号），不用""
- 适当使用 emoji 增加可读性，但不要过多（全文不超过 5 个）

### 短句独立成段标准
- 允许短句，但要“有作用”：
  - 开头情绪反转
  - 关键结论锚点
  - 三连排比中的强调句
  - 转折处的停顿句
- 判定为“碎句”的参考：`<= 8 字` 且信息增量低（仅情绪词/连接词）。
- 密度控制：
  - 1 句独立段（短句段）占比建议 `<= 18%`
  - 连续短句段不超过 2 段，超过则合并
  - 每 300 字建议不超过 3 个短句独立段（确有内容需要可放宽到 4）

### 破折号使用标准
- 用途：插入补充、语义打断、情绪急停。
- 解释说明优先使用 `：` 或 `，`，不优先使用 `——`。
- 密度控制：每 300 字建议 `0-2` 个破折号，超过则优先替换为冒号/逗号。
- 禁止“为断而断”：四五个字频繁独立成句并伴随破折号。

### 俚语/方言使用规则
- 不是每篇都加，只在内容场景需要时启用（生活叙事、人物口述、地域话题）。
- 启用条件：
  - 文章存在明确人物口吻或地域语境
  - 俚语/方言能提升代入感，而非炫技
- 使用上限：建议全文 `1-3` 处；观点型/干货型通常不超过 `2` 处。
- 必须保证可理解：生僻方言首次出现可用同句解释。

### 情绪标点与节奏空格
- 根据情绪强度动态用标点：平叙用「，。」；强调可用「——」「？」；情绪高点可用「！」。
- 允许少量连续标点来保留真人语感（如「？！」「……」），但避免全篇泛滥。
- 允许局部“非完美标点”模拟口语停顿（如短句后直接转折），前提是不影响理解。
- 允许用空行制造呼吸感：关键句前后可单独留白一行。
- 控制规则：每 300 字内情绪标点（`! ? —— ……`）建议 2-6 处，避免机械平均或零波动。

### 标志性怪癖迁移（风格学习模式）
- 在 `--style` 调用时，允许学习并迁移原作者的“标志性怪癖”：
  - 固定口头禅（如“说白了”“你细品”）
  - 标志性标点组合（如“？！”，“——”后短句）
  - 有意保留的“轻微错误标点”与不完美停顿
- 迁移原则：优先保留“像本人”的识别度，其次才是语法完美度。
- 迁移强度：每 300 字可出现 1-3 处怪癖表达，避免整篇失控。
- 风险兜底：若可读性明显下降（阅读断裂、歧义、噪音过高），再回退到标准标点。
- 可读性底线（必须满足）：
  - 连续标点上限：同一处最多 3 个符号（如“？！！”）
  - 错误标点迁移上限：每 500 字不超过 2 处
  - 禁止迁移会改变语义的错误（如把问句改陈述句、句末标点全丢失）

## 四、金句技巧

好的公众号文章需要「金句」来点睛。金句的特征：

### 金句公式
1. **对比式**：A 不是 B，而是 C
   > 努力不是把所有事做完，而是把重要的事做好。

2. **反转式**：表面是 A，其实是 B
   > 你以为的自律，不过是另一种焦虑。

3. **定义式**：所谓 A，就是 B
   > 所谓成长，就是不断和昨天的自己告别。

4. **条件式**：如果 A，那么 B
   > 如果你的计划看起来毫无风险，那它大概也毫无价值。

5. **类比式**：A 就像 B
   > 知识就像压缩包，不解压使用就只是占硬盘。

### 金句放置位置
- **开头**：作为 hook 抓住注意力
- **小标题后第一句**：概括本段核心
- **段落结尾**：总结升华
- **文章结尾**：留下回味

## 五、标题创作技巧

### 开幕雷击法则（核心框架）

爆款标题的核心秘诀是**开幕雷击**——让读者第一眼看到标题就产生点击冲动。

**7 大雷击要素**（标题中至少包含 2 个，越多越好）：

| 要素 | 说明 | 示例 |
|------|------|------|
| **金钱/数字** | 具体金额、数据，给人利益感知 | 「从负债 3000 万到百亿富豪」 |
| **性暗示** | 隔一层纱的暗示，不可露骨 | 「公公半夜的要求」（让读者自己联想） |
| **暴力/冲突** | 对抗、矛盾、戏剧冲突 | 「被连咬三次」「一刀两断」 |
| **死亡/威胁** | 生死、疾病、危险相关 | 「直到查出癌症才明白」 |
| **捷径** | 快速、简单获得结果的暗示 | 「用同一个套路骗了我们 5 次」 |
| **异常/悬念** | 反常识、出乎意料（屡试不爽） | 「店家亏钱反而乐了」「本该伤心却很高兴」 |
| **民族情感** | 爱国、文化认同、集体荣誉 | 「中国人自己的 XX」 |

**关键原则**：
- 异常是最强要素——和正常思维相反的内容，天然让人想点开
- 不直接写暴力/低俗字面，而是「隔着一层纱」让读者自己联想
- 标题公式 = **突出主题 + 矛盾冲突 + 引发悬念**

### 标题润色技巧

1. **植入流量热词**：如「疲惫不堪」「心满意足」「笑容满面」等有画面感的词
2. **用具体、生动的语言**：创造情境、展现角色、反转情节、制造悬念
3. **口语化 > 文艺化**：吸引人的标题看起来「接地气」，不需要文采，需要人味儿
4. **标题和封面是门面**：「没有人有义务透过你平庸的外表，看到你精致的灵魂」

### 高点击率标题公式

1. **数字型**：「学会这 5 个技巧，XX 效率提升 300%」
2. **疑问型**：「为什么你总是坚持不下去？」/ 人物+凭什么+具体事件
3. **对比型**：「月薪 5K 和月薪 5W 的人，差距在哪？」
4. **否定型**：「别再 XX 了，真正有效的方法是……」
5. **身份型**：「30 岁才明白的 XX 个道理」
6. **悬念型（说到一半）**：「如何/为什么 + 事实 + 反转」
7. **痛点型**：「XX 的人，80% 都犯了这个错」
8. **权威稀缺型**：「《人民日报》：XX」「离职前，我把 XX 的干货都供了出来」
9. **金钱+异常组合**：「我 65 岁，再婚老伴每月退休金 9000，直到外孙生日才发现……」
10. **异常+悬念组合**：「知否：梁晗娶墨兰为妻，不是盛祖母面子大，而是……」
11. **圈目标受众型**：标题中直接点名目标人群，让读者觉得"这就是写给我的"。「梨型女生看过来」「35 岁失业危机：中年人该何去何从？」「新手妈妈必看」
12. **实用速成型**：强调零门槛、可直接复用。「0 基础教程」「建议 XX 直接抄作业」「干货满满！公众号运营必备的 10 个工具」
13. **热点借力型**：蹭热点人物/影视/事件，借势获取流量。「Deepseek 火爆全网，XX」「还没看《哪吒》的人，XX」
14. **经验帖型**：身份标签 + 经验分享，增强可信度。「产品经理丨0 经验上岸百度 PM，我的自学教程分享」「哈佛心理学博士也在看的 7 本书」
15. **情绪共鸣型**：直接用情感话题引发共鸣，不需要悬念，靠情绪本身吸引点击。「生了孩子后，和婆婆的关系急速恶化」「我离婚了，因为一只袜子」
16. **矛盾冲突型**：利用反常规行为制造好奇。「为什么我辞掉了年薪 50 万的工作」「一万块搞定装修？」

### 标题长度参考

根据数据分析，不同长度的标题效果差异明显：
- **8-15 个汉字**：最佳长度区间，信息完整且便于快速记忆
- **20 字以内**：安全区间，适合大多数类型
- **20-50 字**：仅适合故事型标题（需要交代人物+冲突+悬念）
- **超过 50 字**：过长，在手机上显示不全，不建议

### 标题自检清单
- [ ] 长度合适（8-15 字最佳，不超过 50 字，故事型可适当放长）
- [ ] 能一眼看懂讲什么
- [ ] 是否包含开幕雷击 7 要素中的至少 2 个
- [ ] 有让人想点开的冲动（异常感、悬念感、利益感）
- [ ] 不是标题党（内容能兑现）
- [ ] 不包含敏感词
- [ ] 没有直接写暴力/血腥/低俗字面
- [ ] 口语化，读出来不拗口

## 六、热点文章写作技巧：龙骨龙鳞法

> 来源于实战手册，专门用于热点型文章的写作框架。

一篇完整的热点文章是一条龙：

- **龙骨**：事件本身（文章梗概），要求把事情讲明白就行
  - 如果事件复杂，只讲一个侧面，不追求全面描述
  - 事件篇幅不宜过长，否则影响完读率
- **龙鳞**：态度和观点
  - 在陈述事件的过程中，一旦情绪上来了，就用词句把它写出来
  - 龙鳞在热点文中只用来站队、表达态度、写清观点
  - 不需要补充过多细节来证明真实性（热点事件本身就是真实的）

**不同难度的热点文**：
- 简单版 = 标题 + 开头 + 事件陈述 + 结论
- 深度版 = 标题 + 开头 + 事件陈述 + 对比论证（可选）+ 结论

## 七、选题的三大特征

好选题必备的三个特征（同时满足越多，爆的概率越大）：

### 1. 选题中要有「异常」
「狗咬人不是新闻，人咬狗才是新闻」——有异常属性的选题自带流量密码。
- 和正常思维相反的事情
- 令人意外的结果或行为
- 注意：不要把异常改成「正常」，否则文章就「顺拐」了，变成廉价鸡汤

### 2. 多篇爆款验证的选题
同一个选题已经被多个账号写出爆款，说明这个选题有流量。

### 3. 让读者有话可说
见仁见智、大众关心且讨论热度高的话题，容易出爆款。
- 不一定要高大上，家长里短也能爆
- 读者能在评论区各抒己见的话题，完读率和互动率都高
