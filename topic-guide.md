# 选题与对标指南

> 选题定生死——好选题是爆款的前提。本文整理了选题方法论、对标账号体系、赛马机制原理，帮助你系统化地找到高潜力选题。

---

## 一、推荐机制：赛马机制

理解平台算法是写出爆款的基础。

### 赛马机制原理

每天有海量文章发布，平台通过「赛马机制」筛选优质内容：

```
你发布文章
  → 系统分发给几百人（初始流量池）
  → 系统根据数据打分（完读率、点赞、转发、评论）
  → 数据好 → 进入更大流量池，与更多文章竞争
  → 再次胜出 → 继续进入更大流量池
  → 循环往复 → 阅读量 5 万 → 10 万 → 100 万
  → 数据不好 → 停留在当前流量池，不再推荐
```

### 关键数据指标

| 指标 | 说明 | 提升方法 |
|------|------|---------|
| **完读率** | 读者读完全文的比例 | 控制字数、设置钩子、不啰嗦 |
| **点赞率** | 点赞数/阅读数 | 引发共鸣、提供价值 |
| **在看率** | 点击「在看」的比例 | 正文中自然引导 |
| **转发率** | 转发数/阅读数 | 内容有社交货币属性 |
| **评论率** | 评论数/阅读数 | 设置争议话题、开放提问 |

### 如何利用赛马机制

1. **寻找蓝海领域**：竞争文章少 → 更容易胜出 → 更快进入大流量池
2. **选择发布时间**：与对标账号错开发布时间（对标 8 点发，你 7 点发）
3. **提升完读率**：开头设置钩子，中间和结尾留下悬念，吸引读者不断往下看
4. **引导互动**：在文中适当引导点赞、收藏、转发（但不能诱导，需自然）
5. **差异化竞争**：与对标爆款做差异化——人无我有，人有我优

## 二、对标账号体系

### 对标账号分级

| 级别 | 特征 | 用途 |
|------|------|------|
| **A 类** | 大量粉丝 + 高互动率 | 学习长期运营策略 |
| **B 类** | 短期出现 10w+ 爆文 | 学习爆款选题和标题 |
| **C 类** | 稳定 2w 左右阅读 | 学习稳定输出方法 |
| **D 类** | 一直起不来的账号 | 避免踩坑、引以为戒 |

### 寻找对标的方法

#### 方法一：微信搜一搜

1. 根据你的赛道想一个**关键词**（如情感领域 → 婆婆、婆媳）
2. 在微信「搜一搜」中搜索该关键词
3. 优先选择 **10w+ 阅读**的文章
4. 进入文章所在的公众号，扒所有爆款标题
5. 从标题中提取更多关键词，继续搜索 → **套娃式获取关键词**

#### 方法二：套娃式关键词发散

```
初始关键词（如：裁员）
  → 搜索找到爆款文章
  → 从爆款标题中提取新关键词（国企央企、欠薪、降薪、亏损）
  → 用新关键词继续搜索
  → 获得更多爆款文章和对标账号
  → 循环往复，源源不断
```

#### 方法三：其他平台

- **脉脉**：职场爆料、大厂信息（适合职场类）
- **知乎**：深度讨论、社会议题（适合观点类）
- **微博**：娱乐热点、实时热搜（适合热点类）
- **豆瓣**：影评剧评、书评（适合剧评/清单类）
- **头条/百家号**：各类频道分类，标题即关键词

### 选择对标爆款文章的标准

1. **阅读量**：首选 10w+，其次 7-8w+
2. **发布时间**：最近 3 个月内（爆过的内容大概率还会爆）
3. **账号注册时间**：最好也是最近 3 个月注册的低粉号（排除粉丝红利因素）

## 三、选题方法论

### 选题的核心逻辑

一篇文章由三部分组成：
- **选题**：文章的骨架，决定文章最终的样子
- **正文**：文章的血肉，填充骨架丰富细节
- **标题**：文章的外衣，决定能否被用户一眼发现

### 六大赛道推荐

| 赛道 | 特点 | 适合人群 |
|------|------|---------|
| 热点类 | 经久不衰，流量大 | 关注时事、反应快的人 |
| 历史类 | 长期深耕，素材丰富 | 对历史有积累的人 |
| 剧评类 | 相对稳定，素材多 | 喜欢追剧看电影的人 |
| 美食类 | 涨粉快，受众广 | 喜欢做饭/吃的人 |
| 职场类 | 话题性强，共鸣度高 | 有职场经历的人 |
| 情感类 | 受众广，爆款频出 | 善于讲故事的人 |

**建议**：选 1 个赛道、2 个细分方向，坚持创作。

### 选赛道的评分法

给自己打分，满分 100（每项 25 分）：

| 维度 | 评估标准 |
|------|---------|
| **感兴趣** | 有意愿日更，没收益也愿意写 |
| **擅长** | 有相关技能和经验，愿意复盘精进 |
| **市场需求** | 供需关系如何，竞争激烈程度 |
| **输出意愿** | 是否愿意改变习惯、持续输出 |

- 60 分以上：可以入手
- 40-60 分：看看哪方面需要克服
- 40 分以下：考虑换赛道

## 四、各赛道选题技巧

### 职场类
- **素材来源**：脉脉（大厂爆料）、知乎、BOSS 直聘
- **爆款方向**：裁员、薪资泄露、职场故事
- **注意**：针对网友评论讨论，不自我想象、不造谣、不攻击企业，保持客观

### 热点类
- **指点江山类**：知乎自然爆款 > 知乎热榜 > 微博热榜（选尖锐的内容）
- **娱乐热点类**：微博有「热」字可选，有「沸」字优先选，有「爆」字要抢速出文
- **养生热点类**：结合节气 + 健康热榜，只往古代习俗靠，不给医学建议

### 剧评类
- **热点+剧评**：追正在热播的剧/电影，趁热度写出多角度影评
- **经典老剧**：《知否》《甄嬛传》等，流量大、信息量大，一部剧能写几百篇

### 历史类
- **追热点历史**：从当下热点找到历史契合点（需高写作功底）
- **中规中矩写法**：选自己喜欢的历史人物/事件，按基调梳理
- **写法**：人物结局反转 / 性格反面 / 事件总结 / 盘点类

### 情感类
- **素材要求**：至少包含以下 8 要素中的 3 个以上：
  曲折性、情绪、信息增量、民生问题、名人、性、暴力、金钱
- **素材来源**：各大平台找有爆款潜质的故事（共鸣、感人、争议、异常）

## 五、选题避坑

### 绝对红线

| 红线 | 说明 |
|------|------|
| **党史军史** | 只能写正面内容，1840 年后中国相关历史慎写 |
| **吃人血馒头** | 不利用逝者做文章、不编造细节、不消费他人痛苦 |
| **财经/健康** | 需要职业认证才能写，无资质不碰 |

### 隐性坑

1. **违禁词**：发文前过「句易网」检测（很多日常词也是违禁词）
2. **标题党嫌疑**：没有违禁词但被限流，检查标题是否夸大
3. **竞品平台关键词**：不要提微博、抖音等竞品名称
4. **企业名称**：用「某通讯公司」「某银行」替代具体企业名
5. **人物命名**：不用大众化真名，用昵称（详见 risk-check.md 人物命名避坑）

## 六、拆解爆款文章模板

找到爆款后，快速拆解的方法：

1. **通读全文**，确定文章核心意思
2. **找转折词**：「但是」「为何」「不过」「然而」标记段落边界
3. **浓缩每段**：把转折词上下的内容各浓缩为一句话
4. **得出模板**：按开头/中间/结尾的结构整理出模板
5. **以自己的话复写**：用自己的语言重新展开

**核心原则**：掌握「拆解思维」比记住具体模板更重要，不同领域的模板不同，但拆解方法通用。
