# wechat-writer —— 微信公众号 AI 写作 Skill

一个有灵魂、会进化的微信公众号内容创作系统。支持五类文章写作、范文风格学习、素材沉淀、质量闸门、发布前合规检查、数据复盘、多平台改写，以及完整的自我进化机制。

## 核心理念

这不是一个"填模板"的写作工具，而是一个**有人格、会成长的写作搭档**。

- **三层人格**：灵魂 DNA（不可变底线）→ 性格系统（缓慢进化）→ 心境日记（每次创作更新）
- **用户输入 ≠ 初稿**：用户提供的文字是"原料"，系统要"做菜"——提取核心观点后用自己的语言重新创作
- **最小追问原则**：用户只需"扔素材 + 说意图"，赛道/读者/字数/风格等全部自动推断
- **质量闸门**：每篇文章必须通过原创度、AI味、人味三项量化检测才能输出
- **记忆闭环**：创作 → 发布 → 效果数据 → 复盘 → 策略调整 → 下次创作更好

## 调用方式与触发关键词

### 一、写文章

**自然语言（推荐）**：
```
写一篇[类型]公众号文章，主题是[主题]
用[风格名]风格写这个主题：[主题]
把这段素材写成[类型]文章
```

**命令式**：
```
/wechat-writer [文章类型] [主题/关键词]
/wechat-writer [文章类型] [主题/关键词] --style [风格名称]
```

**示例**：
```
/wechat-writer 干货 如何高效学习编程
/wechat-writer 观点 AI对教育的影响
/wechat-writer 清单 2024年最值得读的10本书
/wechat-writer 干货 时间管理 --style 犀利派
用某作者风格写一篇观点文，主题是AI教育的误区
```

### 二、风格管理

| 触发关键词 | 功能 |
|-----------|------|
| `风格参考` | 列出当前可用的所有 styles |
| `风格推荐 [主题/素材]` | 返回 Top3 推荐风格 + 理由 |
| `输出格式 [md/json/both/wechat]` | 设置输出格式 |

### 三、范文学习

| 触发关键词 | 功能 |
|-----------|------|
| `学习这篇文章` | 对范文进行完整 8 维度拆解分析 |
| `只学结构` | 只分析文章结构，不分析文风 |
| `只学文风` | 只分析文风语气，不分析结构 |
| `对比风格` | 对比两种已学习的风格差异 |
| `列出所有风格` | 展示已学习的所有风格文件 |
| `删除风格 [名称]` | 删除指定的风格文件 |

### 四、进化系统

| 触发关键词 | 功能 |
|-----------|------|
| `记住这个偏好` | 将当前修改记录为用户偏好 |
| `总结本次经验` | 对本次创作进行复盘总结 |
| `查看我的习惯` | 展示当前的用户习惯档案 |
| `进化报告` | 展示进化日志和经验知识库 |
| `锁定风格` | 冻结人格维度，范文只存 styles/ 不改人格 |
| `解锁风格` | 恢复正常进化模式 |
| `锁定 [维度名]` | 只锁定特定维度（如"锁定犀利度"） |
| `解锁 [维度名]` | 解锁特定维度 |

### 五、自我优化

| 触发关键词 | 功能 |
|-----------|------|
| `自我优化提示词` | 生成下一轮可直接复用的优化提示词包 |
| `自我优化复盘` | 基于最近N篇/一个月/全部，输出策略升级方案 |
| `导出优化方案` | 输出可执行清单 + 提示词 + 成果模板 |

### 六、风险检测

| 触发关键词 | 功能 |
|-----------|------|
| `风险检测` | 对当前文章进行全面风险检查（15章节） |
| `敏感词检查` | 仅检查敏感词 |
| `标题检查` | 仅检查标题风险 |
| `版权检查` | 仅检查版权与原创 |
| `推荐检查` | 仅检查推荐规范 |
| `合规建议` | 给出具体的合规修改建议和安全替换方案 |

### 七、标题学习

| 触发关键词 | 功能 |
|-----------|------|
| `这个标题效果好（附阅读量/打开率）` | 录入标题效果数据 |
| `学习这些爆款标题` | 批量学习标题技巧 |
| `查看标题偏好` | 展示标题偏好画像 |
| `查看高效标题模式` | 展示积累的高效标题模式 |

### 八、选题库管理

### 十五、上传公众号

| 触发关键词 | 功能 | 模式 |
|-----------|------|------|
| `上传公众号` / `发布到草稿箱` / `上传到后台` | 上传到微信公众号草稿箱 | 标准（确认后上传） |
| `帮我上传到公众号` / `发到公众号后台` / `推送到公众号` | 自然语言等价触发 | 标准 |
| `写完直接上传` / `全自动上传` / `一键上传到草稿箱` | 跳过确认直接上传 | 全自动 |
| `重新上传公众号` | 修改后再次上传 | 标准 |
| `预览文章` / `预览一下` / `本地预览` | 生成微信排版 HTML 在浏览器中预览 | 预览（无需 API 凭证） |

### 跨 AI 自然语言等价调用

当平台不支持 `/wechat-writer` 命令时，使用等价自然语言：
```
写一篇[类型]公众号文章，主题是[主题]，风格用[风格名]
学习这篇范文，只学结构/只学文风
基于最近10篇做自我优化复盘，输出可复制提示词
把这篇文章改写成小红书/知乎/短视频口播
```
